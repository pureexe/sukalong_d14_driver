## Useful Tool
Sukalong D14 use Goodix touch sensor same as gpd-win3. So, you can use GPD3 driver as an alternative source but you need to re-calibrate touch 
- [WinGPD3 touch driver](https://github.com/pureexe/sukalong_d14_driver/releases/download/v1.0/Touch.Screen.Driver.WIN3-20201207.zip)
- [WinGPD3 driver installation tool](https://github.com/pureexe/sukalong_d14_driver/releases/download/v1.0/Win.2-3.Touch.pannel.driver.fix.zip)
